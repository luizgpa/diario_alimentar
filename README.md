# Diário Alimentar

App mobile web para registrar refeições com emoções, sensações físicas e níveis de saciedade. Exporta os registros para PDF.

## Stack

- Vue 3 + TypeScript
- Vite
- jsPDF

## Deploy

[Ver demonstração](https://luizgpa.github.io/diario_alimentar/)

## Instalação

```sh
npm install
```

## Desenvolvimento

```sh
npm run dev
```

## Produção

```sh
npm run build
```

## Lint

```sh
npm run lint
```
